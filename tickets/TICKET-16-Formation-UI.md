# TICKET-16: Formation Management UI
**STATUS: BACKLOG**

## Description
Players cannot currently change their Vanguard/Rearguard positions once the run starts. This is a key tactical layer for a digital tabletop game.

## Requirements
- Add a "Reorganize" button to the Inventory Phase.
- Allow dragging/dropping heroes between the Vanguard and Rearguard slots.
- Ensure the updated positions are reflected in the next Battle Phase.
