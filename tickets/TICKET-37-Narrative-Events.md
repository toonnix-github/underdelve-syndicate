# TICKET-37: Narrative Events (The Oracle)
**STATUS: BACKLOG**

## Description
The dungeon currently feels like a series of combat rooms. Narrative "Choice" events add flavor and character depth.

## Requirements
- Create a new interactable type `EVENT`.
- When triggered, open a full-screen or large modal with a "Dark Fantasy" illustration and a short story snippet.
- Offer 2-3 choices (e.g., "Sacrifice blood for a map [Lose 10 HP]", "Pray for health [Heal 20 HP]", "Search the rubble [Find random item]").
