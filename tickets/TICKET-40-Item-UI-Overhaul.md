# TICKET-40: Inventory Cards & Detailed Inspector
**STATUS: BACKLOG**

## Description
The current inventory list is too clinical for a tabletop game. We need to shift to high-fidelity "Item Cards."

## Requirements
- **The Stash Grid**: Instead of a list, display items as square tiles (cards) with their Icon/Illustration.
- **Micro-Stats**: Only show the primary stat boost (e.g., ⚔️+12) on the small card.
- **The Inspector**: Clicking an item tile opens a **Full Portrait Card** (MTG-style).
  - High-res icon.
  - Flavor text.
  - Full stat breakdown.
  - Rarity glow.
  - Passive/Set descriptions.
- **Equipping**: Drag-and-drop or click-to-assign from the Inspector view.
