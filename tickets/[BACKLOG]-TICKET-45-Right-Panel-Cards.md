# TICKET-45: Right Panel Hero Cards (Board Game Aesthetic)
**STATUS: BACKLOG**

## Description
The right dashboard currently displays hero status in a standard UI list style. To reinforce the "Digital Tabletop" aesthetic, we need to revisualize these hero statuses as physical-feeling cards placed on the right panel.

## Requirements
- **Card Aesthetics**: Transform the hero status bars into horizontal or square "Hero Cards" that look like heavy cardstock laid on the table.
- **Visuals**: Maintain vital stats (HP, role) but use textures, subtle drop shadows, and border styles that make them feel tangible.
- **Layout**: Stack or grid the cards naturally within the right dashboard space, ensuring the layout remains clean and responsive.

## Acceptance Criteria
- [ ] Right panel no longer uses generic list UI for heroes.
- [ ] Hero representations are distinct "Cards" with depth/shadows.
- [ ] HP and vital stats are clearly visible in the new card layout.
- [ ] The cards remain visually consistent during damage/healing pulses.
