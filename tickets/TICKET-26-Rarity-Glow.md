# TICKET-26: Rarity Visual Flourish (Glow)
**STATUS: BACKLOG**

## Description
Legendary items should feel legendary.

## Requirements
- Add a persistent "Animated Aura" or glow effect to items of Epic and Legendary rarity in the inventory cache.
- Use a pulsing CSS `box-shadow` or `drop-shadow` matching the rarity color (Purple for Epic, Gold for Legendary).
