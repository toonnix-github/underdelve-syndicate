# TICKET-35: Combat SFX Bundle
**STATUS: BACKLOG**

## Description
The game currently looks great but is silent. We need basic audio feedback to sell the combat impact.

## Requirements
- Source and implement 4 basic audio assets:
  - `hit.mp3`: Short, crunchy impact sound.
  - `heal.mp3`: Shimmering/Magical sound.
  - `move.mp3`: Soft stone-drag or footstep sound.
  - `click.mp3`: Wood/Stone button click for the dashboard.
- Add a "Mute/Unmute" toggle to the dashboard.
