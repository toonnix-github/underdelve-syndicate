# Underdelve Syndicate System Documentation

## 🌓 Core Theme: Premium Dark Fantasy
Underdelve Syndicate is a **Digital Tabletop Tactical RPG** set in a gritty, high-stakes fantasy world. The "Syndicate" is a powerful organization of mercenaries and mages exploring the abyssal ruins of the Underdelve.

### Visual Identity (The "Gothic Anime" Aesthetic)
*   **Art Style:** Highly detailed, stylized anime illustrations with a mature, edgy finish.
*   **Palette:** Deep blacks, dark grays, and stone textures, contrasted with strong elemental glows (Cursed Purple, Sanguine Red, Necrotic Green, and Holy Gold).
*   **Atmosphere:** Moody, dangerous, and "hardcore" fantasy. Materials should feel like ancient metal, treated leather, and bone.

### Visual Asset Guidelines
*   **Item Icons:** Dark backgrounds to ensure item silhouettes pop.
*   **Rarity FX:** Rare+ items feature magical effects (shadows, embers, lightning).
*   **Design Philosophy:** Functional but enchanted. Equipment should look like it belongs to a high-level mercenary group in a dark fantasy setting.

## Core Dictionary

This document defines the strictly enforced Race and Job systems for the Underdelve Syndicate. 

### 1. Races (Origins)
Races determine a unit's biological foundation and provide innate stat modifiers.

| Race | Description | Innate Bonuses |
| :--- | :--- | :--- |
| **Human** | Versatile and adaptable. | +5 HP, +1 SPD |
| **Elf** | Graceful and perceptive. | +3 SPD |
| **Night-Elf** | Shadow-dwelling swiftness. | +5 SPD, -10 HP |
| **Dwarf** | Sturdy and resilient. | +5 DEF |
| **Orc** | Brutish strength. | +4 ATK |
| **Undead** | Relentless and cold. | +2 ATK, +5 DEF, -2 SPD |
| **Beastman** | Primitive agility. | +2 ATK, +2 SPD |
| **Demon** | Corruptive power. | +5 ATK, -5 DEF |
| **Dragon** | Legendary durability. | +10 HP, +5 DEF, -2 SPD |
| **Titan** | Immense stature. | +20 HP, -5 SPD |

### 2. Jobs (Roles)
Jobs define the tactical function of the unit and provide percentage-based attribute scaling.

| Job | Ideal Role | Core Scaling |
| :--- | :--- | :--- |
| **Swordsman** | DPS | +10% ATK |
| **Archer** | DPS | +10% SPD |
| **Priest** | Healer | +15% Healing Power |
| **Knight** | Tank | +15% DEF |
| **Thief** | DPS | +15% SPD, +5% Crit |
| **Mage** | DPS | +15% Magic Damage |
| **Berserker** | DPS | +20% ATK, -10% DEF |
| **Guardian** | Tank | +20% DEF, -5% SPD |
| **Scout** | DPS/Support | +10% SPD, +10% Evasion |
| **Paladin** | Tank/Healer | +10% DEF, +10% Healing |

## Equipment Standards

### Multiplier Rules
1. **Two-Handed Weapons**: Always carry a Speed penalty (-SPD) but offer significantly higher base Attack.
2. **Skill Rarity**: Only **Rare**, **Epic**, and **Legendary** items possess active or passive skills. Common and Uncommon items provide raw stats only.
3. **Job Restrictions**: High-tier equipment is strictly locked to specific Jobs to ensure role purity.
